    几个问题

  • mac 上 gem install 报not permitted错误，是 mac 系统限制导致，指定安装路径到/usr/local/bin/下即可，例如sudo gem install -n /usr/local/bin rdiscount
  • code 的语法高亮，要用到 pygments 及相应的 css
  • git 提交报not raged 错，使用git commit -am "xxx"即可
